Smart To-Do List - Internship Final Project

Project Overview
A professional task management application built with Laravel 12 and Tailwind CSS, designed for efficiency and high usability.

Features 
- Task Management: Full CRUD (Create, Read, Update, Delete).
- Attributes:Tasks include titles, descriptions, due dates, and priority levels (High, Medium, Low).
- Filtering: Functional filters for status (Pending/Completed) and priority.
- Responsiveness: Fully mobile-responsive UI using Tailwind's utility classes.
- Security: Backend validation and route protection.

Technical Implementation
- Backend: Laravel 12 utilizing Models, Controllers, and Migrations.
- Database: Optimized MySQL schema with indexed columns for performance.
- Frontend: Responsive Glassmorphism design using Tailwind CSS
- Version Control: Managed via Git with standardized commit messages.

Assurance Quality 
- Testing: Automated tests using Pest PH
- - Custom Tests: Implemented feature tests for Task access and Security (TaskTest.php).
- Core Tests: 24 infrastructure tests (Auth/Profile) plus 2 custom business logic tests.
